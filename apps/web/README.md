# web

Frontend Next.js do sistema de gestão de projetos. Consulte o [README da raiz do repositório](../../README.md) para instruções completas de setup e variáveis de ambiente.

## Scripts

- `npm run dev` - inicia o frontend em modo desenvolvimento (`http://localhost:3000`)
- `npm run test` - executa os testes de componentes (Vitest)
- `npm run lint` / `npm run check-types` - lint e checagem de tipos
- `npm run build` - build de produção
