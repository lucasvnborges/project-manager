# api

Backend NestJS do sistema de gestão de projetos. Consulte o [README da raiz do repositório](../../README.md) para instruções completas de setup, variáveis de ambiente e documentação da API.

## Scripts

- `npm run start:dev` - inicia a API em modo watch
- `npm run test` - executa os testes unitários
- `npm run test:e2e` - executa os testes de integração
- `npm run build` - compila para `dist/`

Documentação interativa (Swagger) disponível em `/docs` quando a API está em execução.
